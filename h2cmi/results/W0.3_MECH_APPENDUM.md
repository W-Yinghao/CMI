**Terminology (corrected).** `P^{same}` is the **non-transfer decision-prior harm**, not "intrinsic

contains **metric-prior mismatch** (a non-uniform prior is wrong for the balanced-accuracy objective) and
**π_J estimation/calibration error** (e.g. minority-stage underestimation, N1: π_J≈0.005 vs true 0.046).

## Definitions (frozen)

For a unit with evaluation set `E`, let `B_E(π) = balanced-accuracy of the identity-geometry decoder
argmax_y[ log p(z|y) + log π_y ]` on `E`. The decision-prior term is

    P_J = B_E(π_J) − B_E(Unif).

Introduce two **oracle-prior diagnostic** decodings (true prevalences; diagnostic only, never a deployed
predictor): `ρ_E` = true evaluation-set stage prevalence, `ρ_A` = true adaptation-set stage prevalence.
Then the **exact** identity decomposition:

    P_J =  [ B_E(ρ_E) − B_E(Unif) ]        (metric-prior mismatch)
         + [ B_E(ρ_A) − B_E(ρ_E) ]         (adapt→eval prevalence transfer)
         + [ B_E(π_J) − B_E(ρ_A) ]         (π_J estimation / calibration error)

- **W0.3 (same-session, ρ_A = ρ_E):** the transfer term is exactly/numerically 0, so
  `P^{same}_J = metric-mismatch + estimation-error`.
- **W0.1 (cross-night):** all three terms present,
  `P^{cross}_J = metric-mismatch + transfer + estimation-error`.

This is strictly better than reporting only `P^{cross} − P^{same}`: it says whether the −0.11 comes from
using the true non-uniform sleep prevalence under BA (metric-mismatch) or from π_J collapsing N1/REM
beyond the true prevalence (estimation error).

## Aggregation (frozen)

Subject is the biological cluster. Per subject `s`:

    P^{same}_s  = mean over seeds AND over directions {AB, BA}
    P^{cross}_s = mean over seeds (from the frozen W0.1 cross-night run)
    ΔP^{shift}_s = P^{cross}_s − P^{same}_s

Report `mean P^{same}`, `mean P^{cross}`, `mean ΔP^{shift}` with subject-cluster bootstrap CIs (10k). The
magnitude fraction `R = |P^{same}| / |P^{cross}|` is reported with a bootstrap CI and phrased as
"the same-session non-transfer component accounts for roughly X% of the cross-night P magnitude" —
NOT as a causal variance decomposition.
