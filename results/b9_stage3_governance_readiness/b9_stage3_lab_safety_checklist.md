# B9.3 lab safety + recording checklist — DRAFT template (development-only)

> DRAFT to be adapted to the institution's approved lab-safety / recording SOP. Not approval, not legal/IRB advice.

- [ ] Institutional ethics/governance approval obtained (status `approved`) BEFORE any recording.
- [ ] Informed consent completed and documented per approved process.
- [ ] EEG equipment inspected; electrical safety / grounding per lab SOP; no stimulation devices.
- [ ] Impedance / signal-quality checks per SOP before each session.
- [ ] Frozen assignment table + event-marker map loaded read-only into the acquisition app; hash verified.
- [ ] Companion trial log writing verified (subject_slot, microblock, trial_id, C, Y_design, timing, artifact_flag).
- [ ] Participant comfort, breaks between microblocks, stop-on-request honored.
- [ ] Hygiene / electrode handling / cleaning per lab SOP.
- [ ] Adverse-event reporting path per institutional policy.
- [ ] Data written to approved secure storage; no PII in the repository.
