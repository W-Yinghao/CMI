# CSC manuscript — self-contained bundle

**Title.** *When Can Concept Shift Be Certified? From Unlabeled Abstention to Paired Minimal-Label
Certificates.* (A case study in EEG domain shift.) — **Target venue: AAAI-27 main technical track.**

This folder is a **standalone snapshot** of the manuscript: everything needed to build and check the paper,
and nothing else (no research-package code, results, or dev tooling). It is exported from `csc/paper/` in the
CMI repository, which remains the working source of truth. All results are **synthetic**; there is no real-EEG
or clinical claim (real EEG is future work).

## Build

```bash
bash compile.sh        # neutral single-column build  -> main.pdf
bash compile_aaai.sh   # AAAI two-column build         -> aaai_main.pdf (+ page count)
bash check_claims.sh   # wording/scope gate + compile; prints "[N/N checks executed] CHECK_CLAIMS_PASS"
```

Requires a TeX Live install with `pdflatex` + `bibtex`. The neutral build is self-contained (base LaTeX +
`amsmath`, `booktabs`, `tabularx`, `graphicx`, `tikz`, `natbib`, `hyperref`).

**AAAI style note.** `aaai_main.tex` is currently a two-column `article` *approximation* so the paper
compiles and page count is trackable. At submission, drop the official AAAI-27 author-kit files
(`aaai27.sty` + `aaai27.bst`) here and replace the `aaai_main.tex` preamble per the kit. Page compression to
the 7-page main-content limit is a later phase (proofs / large tables / appendix detail move to a supplement).

## Provenance of the numbers

The empirical numbers are locked to two frozen confirmatory runs in the full repository:

- **Route A (negative)** — tag `csc-confirmatory-v1` / `dee8958`; `confirmatory.json`. Forbidden 1/65
  (CP-upper 0.0709 > α); power 0.43 (CP-lower 0.326); `headline_core_pass=false`.
- **Route B3 (positive)** — tag `csc-b3-confirmatory-v1` / `0595f64`; `b3_confirmatory_result.json`.
  C1–C6 pass: worst control CP-upper 0.0267 < α; primary power 1.000 (per-cell CP-lower 0.985); C6
  independent red-team reproduced the verdict without correction.

Fig 3 was generated by `csc/tools/make_fig3_evidence.py` (in the full repo), which loads and asserts every
number against those two artifacts. The two figures are included here as raster/vector assets so the bundle
builds without the full repository.
