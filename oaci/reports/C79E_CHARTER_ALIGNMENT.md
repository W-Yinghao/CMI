# C79E Charter Alignment

## Status

This is an additive interpretation-only alignment. It changes no scientific
registry entry, estimand, threshold, model, kernel, null, multiplicity family,
RNG stream, execution path, or report order.

The locked C79P object is situated in the project charter as follows:

```text
program object:
  Information-Constrained Model Selection under Conditional Distribution Shift,
  Group-Dependent Transport, and Extreme Candidate Selection

C79E role:
  R1 training-seed robustness only

P1 role:
  minimal information sufficiency and extreme-action utility

P2 role:
  local association plus conditional transport audit

H2R/H4R/H5R/H6R:
  mandatory secondary registered candidate audits
```

Evidence levels remain separate:

```text
L1 reliability   = repeated measurement stability
L2 association   = registered dependence without predictive qualification
L3 prediction    = held-out incremental endpoint prediction
L4 transport     = held-target and held-regime qualification
L5 actionability = material top-k, regret, or coverage control
```

No lower level implies a higher level. Seed 4 perturbs learned parameters and
optimizer trajectories while retaining the same targets and raw trials; it does
not establish new-subject, target-population, dataset, or external generality.
