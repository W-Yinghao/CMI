# C84SL Label-Frontier Stability Timing Audit

The scientific protocol already assigns `C84-L3` when registered dataset,
level, or target-composition heterogeneity is active. This no-label
clarification makes the level and LOTO clauses executable before the C84S
implementation commit and before any target-label access.

Level-specific frontiers use the unchanged target-cluster qualification and
closure. A level mismatch is active when the two level-specific `B*` values are
not identical. Target-composition preservation requires the LOTO `B*` to equal
the full-panel dataset `B*`; the unchanged 75% thresholds are 17/22, 15/20 and
57/76. No method, budget, margin, draw count, target, or taxonomy precedence is
changed.
