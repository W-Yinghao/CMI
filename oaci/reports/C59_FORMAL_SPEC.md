# C59 - Formal Spec

C59 defines selectors as measurable maps from an information class I0-I7 to one candidate per cell. Empirical metrics remain finite-population diagnostics unless a theorem statement and assumptions are explicitly written.

- I0_random_or_tie: cell candidate set only -> baseline
- I1_strict_source_observables: X_s source metrics -> weak signal, not sufficient
- I2_source_plus_keys: X_s plus K -> key-only escape closed
- I3_target_unlabeled_geometry: X_s, K, U0 -> not available for closure in committed cache
- I4_target_grouped_zero_label_structure: target/trajectory grouping -> diagnostic ceiling/actionability split
- I5_split_label_or_few_label: disjoint target-label cache -> future unresolved
- I6_target_label_diagnostic_content: D and cross-cell templates -> diagnostic partial closure
- I7_same_label_endpoint_oracle: E same-candidate endpoint scalar -> oracle reference only
