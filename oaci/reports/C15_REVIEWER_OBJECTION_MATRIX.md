## 2. Over-claim guardrails (recommended wording fixes)

- **C1_measurement →** "Under domain-class support mismatch, conditional-invariance diagnostics that smooth over cells with no source support are not well-identified; we define a support-aware, source-only leakage diagnostic (estimable vs unsupported cells, grouped cross-fit L_Q^ov, grouped permutation null) and show on BNCI2014-001 LOSO (ShallowConvNet, seeds 0-2, one fixed probe family) that it yields reproducible, source-isolated, auditable measurements — a null held-out audit signal (0/54 BH-FDR survivors) alongside a large measured selection-time reduction (Δsel -0.326). We do not claim to have demonstrated the failure of a naive estimator or quantified the support mismatch on this dataset; the measurement is descriptive and probe-relative."
- **C2_control_failure_localization →** "On BNCI2014-001 LOSO (ShallowConvNet, seeds 0-2, 54 fold-levels), the battery localizes OACI's control failure: selection-time leakage reductions do not transfer to the held-out source audit (Δsel -0.326, reduced 54/54, vs Δaudit +0.008, reduced 25/54, corr +0.004); nominal per-fold audit signals do not survive multiplicity (11 nominal, 0 Bonferroni, 0 BH of 54); the pre-registered K2 endpoint check returns stop_no_reproducible_gain (worst-domain bAcc 2/6 improved, worst-domain NLL improved on average but not reproducibly, worst-fold +0.32); and a diagnostic, non-deployable source-AUDIT oracle — which reads held-out source_audit but never the target — cannot identify a gain-reproducing checkpoint (C_oracle_also_fails; replay 216/216, 0 argmax flips). This localizes the failure of the tested source-side control under support mismatch; it is NOT evidence that DG, EEG transfer, or a target-informed selector fails, nor that it generalizes beyond this dataset family and backbone."
- **C3_anti_transfer →** "On BNCI2014-001 LOSO (ShallowConvNet, seed 0), across all 6 actively-trained cells (3 targets x 2 tau_lse), the SRC source-endpoint objective drove source guard NLL down by ~1 nat (to near-zero ~0.09, consistent with guard memorization) while worsening target worst-domain NLL in every one of the 6 cells (ATI 1.0 over these 6 cells; source→target NLL rank-anticorrelated, Spearman -1.0, n=6). SRC-caused target-NLL blowups occur in 4/6 active cells (2 further of the '6/12' blowups are ERM-fallback cells whose NLL already exceeds uniform, not SRC's doing), and target 3 shows only marginal NLL anti-transfer with a small accuracy gain at tau=0.1. We do not claim this generalizes beyond this dataset family, backbone, or seed, nor to better-regularized source objectives; it is a stress replication showing this source-side control signal does not transfer."
- **C4_framework →** "The reusable contribution is a falsification battery (G0 integrity, G1 selection-optimism, G2 held-out-leakage K1, G3 endpoint-transfer K2, G4 source-audit-oracle replay, G5 source→target anti-transfer), which we INSTANTIATE ONCE — BNCI2014-001 LOSO with ShallowConvNet, seeds 0-2 for K1/K2 and seed 0 for the anti-transfer stress replication — to FALSIFY two source-side control mechanisms (OACI, SRC). Under this single protocol the verdict is 'control hypothesis falsified' [no reproducible endpoint transfer, source-audit oracle fails to rescue, source→target anti-transfer at seed 0]; on this basis we close OACI-v2 and SRC as control objectives UNDER THIS PROTOCOL (not as a general impossibility) and retain support-aware leakage as measurement. We do not claim the battery is validated across datasets, backbones, or that any DG penalty must fail; generality of the instrument is future work."

## 3. Genuine evidence gaps (future work, NOT current claims)

- **Support-mismatch existence unquantified (C1):** no committed count of estimable vs unsupported domain-class cells for BNCI2014-001; only `audit_support_hash` is committed, so whether any support mismatch actually exists in this dataset is unshown.
- **No naive-vs-support-aware / ungrouped-vs-grouped contrast (C1):** "ill-posed" is a premise, unbacked by any committed number showing a naive estimator is biased/anti-conservative/undefined.
- **Probe-relativity unquantified (C1):** sensitivity of L_Q^ov to the probe-capacity family and reference prior — fixed by protocol, never varied.
- **No second dataset that genuinely exhibits support mismatch (C1/C2/C4):** BNCI2014-004 explicitly NOT run (pre-registered pause); e.g. an imbalanced clinical cohort confirming support-awareness changes the answer.
- **No target-side oracle replay (C2):** C10b uses only a source-audit oracle (S5, reads_target=False), so "no good checkpoint exists in the trajectory" is established only w.r.t. held-out source signal, not absolutely.
- **No uncertainty on K2 endpoint split (C2):** worst_domain_bacc 2/6 improved has no CI or p-value; margin=0.0, n_units=6; only the 3-seed K2 minimum, not the referenced 5-seed manifest.
- **Multi-seed SRC anti-transfer (C3/C4):** C12/C3 is seed-0 only; ATI=1.0, STI=1.0, pearson -0.947 all rest on n=6 single-seed cells with no CI.
- **SRC regularization / lambda / lr sweep (C3):** source guard NLL collapses to ~0.09 (memorization); only tau_lse in {0.1,0.3} varied, so anti-transfer under a well-regularized SRC is untested (deepest hole).
- **Target-selection rationale (C3):** only 3/9 targets ([1,3,5]) with no committed rationale; target 3 anti-transfers only marginally.
- **Second backbone (C1-C4):** only ShallowConvNet instantiated; nothing shows the measurements or verdicts are backbone-invariant.
- **Full 5-seed K1/K2 manifest (C2/C4):** C8 runs minimum seeds [0,1,2] and self-labels "not yet the full 5-seed manifest sweep."
- **Positive control / discriminative validity (C4):** no committed non-falsified verdict; no ERM-beating DG baseline run through the same battery to show the gates discriminate transfer from anti-transfer rather than always failing.
- **Stronger baselines (C2):** ERM, global_lpc, uniform only; no strong tuned DG competitor to show ERM-parity is a meaningful bar.

## 4. Overall verdict

All four claims survive scrutiny (overclaim_risk = minor each) and are defensible ONLY with the recommended wording fixes applied — every headline must be explicitly scoped to BNCI2014-001 LOSO / ShallowConvNet (C3/G5 additionally to seed 0, n=6) and framed as "support-aware leakage measurement retained, source-side control (OACI, SRC) falsified under this protocol — NOT a claim that DG / EEG transfer / target-informed selection fails."
